Damit AIs von der Software geladen werden können, müssen die folgenden Bedingungen erfüllt werden:
	- es muss sich bei der AI um eine .class-Datei handeln
	- die AI muss das User Interface implementieren, welches sich im /ai Ordner befindet
	- der Klassenname der AI muss dem Dateinamen der .class-Datei entsprechen
	- die AI muss das Package "ai" haben
	- die AI muss sich in einem Ordner namens "ai" befinden

Werden alle diese Bedingungen erfüllt, so können AIs beliebig geladen werden.
